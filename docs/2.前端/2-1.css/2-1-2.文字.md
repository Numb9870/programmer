# 文字

## 文字边框

### 实现方式1：文字阴影
> - 缺点：
>   1. 不能显示背景颜色。
>   2. 仅8个方向有阴影，会产生锯齿。

<WordShadow1 />

:::details 代码
```vue{17-20}
<template>
  <div class="container">
    <p class="word-shadow">Word Shadow</p>
  </div>
</template>

<style lang="scss" scoped>
.container {
  width: 100%;
  height: 200px;
  background-color: #333;
  display: flex;
  justify-content: center;
  align-items: center;
  .word-shadow {
    font-size: 5rem;
    // 设置文字跟背景颜色一致
    color: #333;
    // 设置8个方向的文字阴影
    text-shadow: #fff 1px 0, #fff 1px 1px, #fff 0 1px, #fff -1px 1px, #fff -1px 0, #fff -1px -1px, #fff 0 -1px, #fff 1px -1px;
  }
}
</style>
```
:::

### 实现方式2：-webkit-text-stroke
> - 优点：文字可以显示背景颜色
> - 缺点：
>   1. 兼容性问题
>   2. 描边是居中描边，且无法更改。（可以使用::after 设置一样大小的文字内容去覆盖）

<WordShadow2 />

:::details 代码
```vue{17-27}
<template>
  <div class="container">
    <p class="word-shadow" data-content="Word Shadow">Word Shadow</p>
  </div>
</template>

<style lang="scss" scoped>
.container {
  width: 100%;
  height: 200px;
  background: linear-gradient(to right, blue, pink);
  display: flex;
  justify-content: center;
  align-items: center;
  .word-shadow {
    font-size: 5rem;
    // 设置文字颜色为透明
    color: transparent;
    // 设置文字边框
    -webkit-text-stroke: 2px #fff;
    // 覆盖文字内容，防止文字变细
    &::after {
        content: attr(data-content);
        position: absolute;
        left: 0;
        top: 0;
        -webkit-text-stroke: 0;
    }
  }
}
</style>
```
