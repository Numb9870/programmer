# 分支

## 单分支

> 语句前面必须要有空格

```python
# 单分支
age = 18
if age >= 18:
    print("成年人")
```

## 双分支

```python
# 双分支
age = 18
if age >= 18:
    print("成年人")
else:
    print("未成年人")
```

## 多分支

```python
# 多分支
age = 18
if age >= 18:
    print("成年人")
elif age >= 12:
    print("青少年")
else:
    print("未成年人")
```

## 嵌套分支

```python
# 嵌套分支
name = "王五"
name = "李四"
age = 18
gender = "男"
if name == "王五":
    if age >= 18:
        print("王五是成年人")
    else:
        print("王五是未成年人")
elif name == "李四":
    if gender == "男":
        print("李四是男人")
    else:
        print("李四是女人")
else:
    print("不是王五也不是李四")
```
