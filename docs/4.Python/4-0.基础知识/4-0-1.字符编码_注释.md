# 字符编码—_注释

## 字符编码

> - 字符编码是指将字符映射为二进制数字的规则。
> - 不同的字符编码方案使用不同的数字来表示不同的字符。
> - 常见的字符编码方案有 ASCII、UTF-8、GBK 等。
> - 默认情况下，Python 3 中使用的字符编码是 UTF-8。

```python
#coding=utf-8
```

## 字符编码转换

> - 可以使用 `ord()` 方法将字符串转换为 Unicode 编码。
> - 可以使用 `chr()` 方法将 Unicode 编码转换为字符。

```python
# 字符编码转换示例
print(ord("a")) # 97
print(ord("你")) # 20320

print(chr(97)) # a
print(chr(20320)) # 你
```



## 单行注释

> - 单行注释以 # 开头，# 后面的内容为注释内容

```python
# 这是一个单行注释
print("hello world")
```

## 多行注释

> - 多行注释以三个单引号或三个双引号开头和结尾，注释内容可以跨越多行。

```python
'''
这是一个多行注释
可以跨越多行
'''
```
