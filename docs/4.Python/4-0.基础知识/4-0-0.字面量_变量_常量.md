# 字面量_变量_常量

## 字面量

> - 写在 Python 文件头部的字符串，会被自动识别成 docstring(文档字符串)
> 
> - 所谓字面量，就是直接写在代码中的具体值。
> 
> - Python 的字符串中可以包含任意字符，且必须使用引号包起来。
>
> - 字符串的引号，可以是:单引号、双引号、三个单引号、三个双引号。
>
> - 文档字符串的主要作用是:对当前 Python 文件进行一些说明，且文档字符串必须用三个双引号包起来。

## 变量

> - 变量是用来存储数据的容器，每个变量都有一个唯一的名称，用于引用存储在其中的数据。
> 
> - 当我们想读取或修改某个值的时候，可以通过变量去完成读取和修改操作


## 常量

> - 常量是指在程序运行过程中，其值不会改变的量。
> 
> - Python中一般约定使用全大写变量名来表示常量，涉及到多个单词时用下划线做分隔
>
> - Python中没有强制的常量机制，Python中所谓的常量，其本质其实还是变量，只不过我们约定好不去修改，

## 代码展示
```python
# 字面量
张三
18

# 变量
name = '张三'
gender = "男"
hobby = '''篮球'''
hat = """黑色"""
age = 18

# 变量命名规则
## 小驼峰(lowerCamelCase)
userName = '张三'
## 大驼峰(upperCamelCase)
UserName = '张三'
## 蛇形(snake_case)
user_name = '张三'

# 常量
PI = 3.1415926
```
